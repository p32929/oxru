# Bundled fonts

These fonts are bundled with Oxru and loaded from this folder (never from the
system) so the windowed (`--gui`) mode renders text and icons consistently
regardless of the host environment.

## JuliaMono.ttf
**JuliaMono** by cormullion — the text font, chosen for its very large Unicode
coverage (box-drawing, arrows, and the macOS keyboard glyphs ⌃⇧⌥⌘ used to render
shortcuts).

- License: SIL Open Font License 1.1 (OFL-1.1)
- Upstream: https://github.com/cormullion/juliamono

## SymbolsNerdFontMono.ttf
**Symbols Nerd Font Mono** from the Nerd Fonts project — supplies the icon
glyphs (file-type, folder, UI) as a fallback font.

- License: MIT (Nerd Fonts project); bundled glyphs retain their upstream
  licenses (OFL / MIT / Apache).
- Upstream: https://github.com/ryanoasis/nerd-fonts

Both are redistributed unmodified under licenses that permit bundling.
